g.raphael-radar by Valve

Radar chart implemented with Raphael.js

Works where Raphael.js works (Firefox 3.0+, Safari 3.0+, Chrome 5.0+, Opera 9.5+ and Internet Explorer 6.0+). Only 2.35 KB (1.02KB gzipped) -very lightweight!

Dependencies

Raphael.js

Install

var paper = Raphael('some_div_id', 800, 800); paper.radar(cx, cy, radius, data_array, options);
